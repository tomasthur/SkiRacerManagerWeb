## Ski Racer Manager – Web

Prezentacný web pre mobilnú hru Ski Racer Manager.

### Požiadavky
- Node 18+

### Inštalácia a spustenie
```bash
npm install
npm run dev
# otvorte http://localhost:3000
```

### Technológie
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion

### Farby témy
- lightBlue: `#f0f1f5`
- mediumBlue: `#2f8ce7`
- darkBlue: `#264e86`
- orange: `#ffa500`

Obrázky a videá doplníme neskôr – komponenty sú pripravené.
